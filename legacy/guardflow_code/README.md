# Legacy GuardFlow / SEVE-Universal Codebase

Este diretório contém uma cópia do framework **SEVE-UNIVERSAL** original,
associado ao projeto GuardFlow, mantida apenas para referência histórica.

Conteúdos incluídos:

- SEVE-UNIVERSAL/src/seve_universal/: Implementações antigas de core,
  adapters, empathy, ethics e integrações v3.
- SEVE-UNIVERSAL/examples/: Exemplos práticos do legado.
- SEVE-UNIVERSAL/tests/: Testes de sanidade.
- Metadados como README.md, README_v3.md, CITATION.cff, licenças e
  pyproject.toml.

> Nota: O diretório _git_legacy/ é apenas um snapshot do repositório original
> e não deve ser utilizado como controle de versão dentro deste projeto.

Todo o desenvolvimento ativo do SEVE Framework continua nos diretórios
principais (src/seve_framework/, docs/, scripts/, etc.).
